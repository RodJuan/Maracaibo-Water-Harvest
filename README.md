# Proyecto Maracaibo: Cosecha Atmosférica de Agua (AWG) v1.1

## 🌐 Visión General
¿Es posible satisfacer las necesidades de agua de una población urbana utilizando únicamente la humedad del aire y física pasiva? 

Este proyecto investiga la viabilidad de la **Generación Atmosférica de Agua (AWG)** en la cuenca del Lago de Maracaibo, Zulia, una de las zonas con mayor humedad relativa (HR) y radiación UV del planeta. El objetivo es diseñar un sistema de bajo costo, escalable y soberano que transforme la humedad ambiental en agua líquida sin dependencia de la red eléctrica.

## 🏗️ Especificaciones del Prototipo v1.1
Basado en una auditoría forense de termodinámica, el diseño actual se enfoca en:
- **Dimensiones:** Marco de 1m x 1m.
- **Material:** Malla HDPE Raschel (80% sombra) con estabilización UV premium.
- **Técnica de Nucleación:** Doble capa con tensión mecánica optimizada.
- **Hack Termodinámico:** Integración de enfriamiento radiativo pasivo (Emisividad IR 8–13 μm) para alcanzar el punto de rocío de forma autónoma.

## 📊 Metodología de Validación
El sistema utiliza un nodo de monitoreo remoto (NemoClaw Node) para registrar:
1. Temperatura y Humedad Relativa (Sensores DHT22/SHT3x).
2. Tasa de condensación real vs. teórica.
3. Degradación de materiales bajo exposición solar extrema.
