# Especificaciones Técnicas y Termodinámicas

## 1. Parámetros de Diseño
- **Ángulo de Inclinación:** 22° respecto a la horizontal (optimizado para flujo de gravedad y captura de brisa).
- **Malla:** Polietileno de alta densidad (HDPE) con recubrimiento hidrofóbico.
- **Estructura:** PVC de presión 1" con refuerzos estructurales de 45°.

## 2. Variables de Entorno (Maracaibo)
- **Humedad Relativa Promedio:** 75% - 85% (Nocturna).
- **Temperatura Nocturna:** 26°C - 28°C.
- **Punto de Rocío Teórico:** ~22.3°C.
- **Delta T Requerido:** ~4°C de enfriamiento pasivo sobre la superficie de la malla.

## 3. Lista de Materiales (BOM)
- [ ] 2x Tubos PVC 1" (6m).
- [ ] 10x Conectores PVC (Codos/T).
- [ ] 2m Malla Raschel 80% (UV Stabilized).
- [ ] Pintura de alta emisividad térmica (Blanco Titanio o recubrimientos IR).
- [ ] Canaleta de recolección de 1.1m.
- [ ] Sensor DHT22 + Microcontrolador (ESP32/Raspberry Pi).
